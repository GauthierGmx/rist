Arborescence des fichiers :

À la racine du dépôt, il y a un dossier par compétence :

- developpement (C1-C2)
Arborescence du dossier developpement :
* algorithme - Contient l'algorithme
* categorie - Contient les dictionnaires des catégories, dans des fichier py
* docs - Contient la documentation Doxygen
* specification - Contient les specifications externes du problème algorithme
* src - Contien le code source

- reseau (C3)
- gestion_donnees (C4)
- gestion_projet (C5-C6)

Nous ferons évolué l'arborescence de ses dossiers au fur et à mesure de l'avancemen du projet.
